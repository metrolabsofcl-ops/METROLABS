☁ Cloud sync with Google Sheets

Connect this app to a Google Sheet so your data is available on any device — phone, tablet, or another computer. Set it up once (about 10–15 minutes), then just click "Sync now" whenever you want to push or pull data.

📋 Setup guide — one time only

  1. Go to sheets.google.com and create a new blank spreadsheet. Name it "MetroLabs Data".
  2. In the sheet, click Extensions → Apps Script.
  3. Delete anything in the code editor, then paste the script Claude gave you (see chat message).
  4. Click Deploy → New deployment. Choose type "Web app".
  5. Set "Execute as" = Me, and "Who has access" = Anyone. Click Deploy.
  6. Authorize the permissions when prompted (click Advanced → Go to project if warned).
  7. Copy the Web app URL it gives you, paste it into the box above, and click "Save URL".
  8. Click "Push local data to Sheet" once to upload your existing bookings.
  9. On any other device, open this same HTML file, paste the same URL, and click "Pull data from Sheet".

This uses your own free Google account — no third party sees your data. You control the Sheet.

Data is saved in this browser's local storage, and can be synced to your own Google Sheet (see ☁ Sync tab) for use on multiple devices.
